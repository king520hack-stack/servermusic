O pacote de recursos do servidor para ouvir músicas incríveis!

📥 Baixar Pacote Java

Como instalar manualmente:

  1. Clique no botão acima para baixar o arquivo aurun_music.zip.
  2. Abra o Minecraft Java.
  3. Vá em Opções... > Pacotes de Recursos...
  4. Clique em Abrir pasta de pacotes.
  5. Arraste o arquivo baixado para dentro desta pasta.
  6. Feche a pasta e ative o AurunMusic na lista do jogo!
